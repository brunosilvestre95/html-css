    <!--

	HIERARQUIA DE TABLEAS (SIMPLES)
        	TABLE = tabela
            		TABLE ROW = Linha de tabela
                		TABLE HEADER =  Cabeçalho de tabela
                    			TABLE DATA = dado de tabela 

	**Se a hierarquia da tabela não for seguida,
		o dado que foi digitado aparecerá ao lado de fora e antes ou depois da tabela


Para estilizar uma tabela é usado CSS

body {
	font-family: Arial }

table {
	width: 500px }

td, th {
	border-style: 1px solid black }
	*É PRECISO SEPARAR TD E TH POR UMA VÍRGULA*
												-->

usar subclasse para cada uma linha ter uma cor de fundo diferente
ex
	        tbody > tr:nth-child(2n) {
            background-color: lightgray;
        }
