		*EFEITO ZEBRADO*


usar subclasse para cada uma linha ter uma cor de fundo diferente
ex
	        tbody > tr:nth-child(2n) {
            background-color: lightgray;
        }

[toda tr que for filha de tbody, a cada linha tera uma cor diferente]

outro exemplo:

			efeito zebrado, linha par terá uma cor,
						linha ímpar tera uma cor diferente
			
	tbody > tr:nth-child(odd) {
		background-color: white;
	
	tbody > tr:nth-child(even) {
		background-color: lightgray;


		**efeito de manter titulo da coluna em tabelas muito longas**
	/* adicionando position stick para toda */

		position: stick; }

		top:0;

 			**mesclagem de células**

	para mesclagem de coluna
			adicionando COLDSPAN="[nº de qtd de células]" dentro da tag
		ex:
			<td colspan="2">[TEXTO CONTEÚDO]</td>
	
	para mesclagem de célula na linha
			adicionando ROWSPAN="[nº de qtd de células]" dentro da tag
		ex:
			<td rowspan="2">[TEXTO CONTEÚDO]</td>

			** Escopo de Grupos **

existem 4 tipos de escope group

	escope="col"
				grupo de uma coluna

	escope="colgroup"
				grupo de varias colunas
	escop="row"
				grupo dee linha
	escope="rowgroup" 
				grupo de linhas

	** o tfoot, mesmo sendo um total ele também é um "row" um" **

 			**AGRUPANDO COLUNAS COM COLGROUP**

	criando a tag <colgroup> e dentro adicionando as tags "col class" desejadas
	usando "<col class="[nome para o grupo da coluna">
		é possível usar nas configuraçoes da css
	"col.[nome-da-coluna] { }	
		também é possível adicionar mais de uma coluna separando por vírgula
	"col.[nome-da-coluna], [nome-da-coluna] { }

			EXEMPLO:
		CSS:
        col.cnome {
            background-color: chartreuse;
        }

        col.csexo, col.cidade {
            background-color: yellow;
        }

		HTML:
	
        <colgroup>
        <col class="cnome">
        <col class="csexo">
        <col class="cidade">
        <col class="cprof">
        </colgroup>
        <caption>Cadastro</caption>
        <tr>
            <th scope="col">Nome</th>
            <th scope="col">Sexo</th>
            <th scope="col">Idade</th>
            <th scope="col">Profissão</th>
        </tr>
        <tr>
            <td>Ana Maria</td>
            <td>F</td>
            <td>25</td>
            <td>Jornalista</td>
        </tr>


	== TAMBÉM É POSSÍVEL EM UMA COLUNA APLICAR A MSM CONFIGURAÇOES NAS ADJACENTES A SEGUIR ==
