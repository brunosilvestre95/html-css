"Marcadas para reutilização com modificação" é um direito maior de usar imagens que deixaram você usar e alterar, inclusive comercialmente, apenas vai exigir mais buscas;
"Marcadas para reutilização" você pode usar comercialmente, pessoalmente;
"Marcadas para reutilização não comercial com modificação"Você pode usar e modificar, mas não para ganhar dinheiro.
